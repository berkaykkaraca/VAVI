About VAVI

The aim of the VAVI project is to empower visually impaired individuals by providing them with real-time environmental awareness through artificial intelligence and smart device technologies. The system utilizes a smartphone camera to capture live images or videos, which are transmitted to a local server for advanced processing using deep learning models, such as YOLO. The server analyzes the visual data to detect people and potential obstacles, and then delivers intuitive audio feedback—either as directional beeps or spoken instructions—back to the user via the mobile device. By enabling users to better perceive their surroundings and navigate safely, VAVI represents a significant step toward increasing independence, safety, and confidence for visually impaired people. The project is currently in the prototype phase, with ongoing development aimed at enhancing features, security, and usability.
